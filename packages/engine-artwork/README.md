# Engine Artwork

This package contains the source artwork files for the Guardians of Humanity project. This includes, but is not limited to:

-   Blender models (`.blend`)
-   Illustrator files (`.ai`)
-   Photoshop files (`.psd`)
-   Other assets used in the creation of game visuals.

These are the raw, editable assets. For optimized, game-ready assets, see the `@engine-assets` package.
